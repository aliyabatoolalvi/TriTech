Conditions

In Java, conditions are implemented using conditional statements. The most common types of conditional statements are if, else if, else, and the switch statement. These statements allow you to control the flow of your program based on certain conditions.

If statement

int number = 10; if (number > 0) { System.out.println("The number is positive."); }

if-else statement

int number = -5; if (number > 0) { System.out.println("The number is positive."); } else { System.out.println("The number is non-positive."); }

if-else if-else statement

int number = 0;
if (number > 0) {
System.out.println("The number is positive.");
} else if (number < 0) { System.out.println("The number is negative."); } else { System.out.println("The number
is zero."); }


Nestes if statement

int x = 5;
int y = 10;
if (x > 0) {
if (y > 0) {
System.out.println("Both x and y are positive.");
} else {
System.out.println("Only x is positive.");
}
} else {
System.out.println("Neither x nor y is positive.");
}

switch statement

int dayOfWeek = 2;
String day;
switch (dayOfWeek) {
case 1:
day = "Sunday";
break;
case 2:
day = "Monday";
break;
case 3:
day = "Tuesday";
break;
// ... other cases ...
default:
day = "Invalid day";
}
System.out.println("Day of the week: " + day);
The switch statement is used when you have multiple possible values to compare against a single variable.
